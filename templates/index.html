About This Application

This application demonstrates how to use a SQLite database with FastAPI and synchronize changes back to a GitHub repository.

How It Works

  1. The application uses SQLAlchemy with SQLite for data storage
  2. Changes made to the database are stored locally
  3. When you click "Sync with GitHub", the database is uploaded to your GitHub repository
  4. This allows your data to persist between deployments on Render
